# UI 素材索引

## 说明

本目录用于存放可人工维护的 UI 参考素材。

原则：

- 文件名使用中文，便于人工检索
- 每个素材都记录来源
- 优先保存公开官方素材或公开商店截图
- 对于无法直接收录的内容，至少保留来源链接和用途说明

## 维护建议

- 更新素材时，尽量保留旧版本并在文件名中追加日期
- 如果某张图只用于局部控件对齐，可在旁边补一张手工裁剪版本
- 若后续加入更多平台，建议继续按“平台/来源/中文语义名”扩展
